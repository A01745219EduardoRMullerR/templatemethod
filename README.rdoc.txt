== Overview

The Template Method pattern is a program that takes data and returns a string formating that data as CSV, HTML and ASCII tables.

Original files and CSV class were already done as example by professor Ariel Ortiz

The UML diagram for this programming activity is as follows:

rdoc-image:img/

To test the program, inside the +src+ folder type the following instruction at the command line:

    ruby table_generator_test.rb
